# Light / Dark Theme Toggle (DOM)

## Description
This project demonstrates a Light and Dark Theme toggle using JavaScript DOM manipulation.

## Features
- Toggle between Light and Dark themes
- Dynamic button text change
- Smooth UI transition

## What is DOM?
DOM (Document Object Model) allows JavaScript to interact with HTML and CSS.
Using DOM, we can:
- Select elements
- Modify styles
- Handle user events

## DOM Usage in This Project
- `document.getElementById()` selects the toggle button.
- `classList.toggle()` adds or removes the dark theme class.
- `addEventListener()` handles button click events.

## Technologies Used
- HTML
- CSS
- JavaScript (DOM)
