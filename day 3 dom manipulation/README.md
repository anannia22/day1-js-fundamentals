# Counter App (DOM Manipulation)

## Description
This is a simple Counter App built using HTML, CSS, and JavaScript.  
It demonstrates how to use **DOM (Document Object Model)** manipulation to update web page content dynamically.

## Features
- Increment the counter
- Decrement the counter
- Reset the counter to zero

## What is DOM?
DOM stands for **Document Object Model**.  
It represents the HTML document as a tree structure where JavaScript can:
- Access HTML elements
- Modify content
- Change styles
- Handle events

## How DOM is Used in This Project
- `document.getElementById()` is used to select HTML elements.
- `addEventListener()` listens for button click events.
- `textContent` updates the counter value dynamically without refreshing the page.

## Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation)
